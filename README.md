# CHAOS LAB

**Interactive dynamical systems explorer** — runs entirely in the browser, zero dependencies.

🔴 **[→ ABRIR CHAOS LAB](https://digitaldrqro-cloud.github.io/chaos-lab)**

---

## Qué es

Un laboratorio de sistemas dinámicos caóticos construido en Canvas + JavaScript puro.
Simula 4 atractores clásicos usando integración RK4 de cuarto orden.

## Sistemas incluidos

| Sistema | Parámetros | Aplicación |
|---|---|---|
| Lorenz | σ, ρ, β | Clima, turbulencia, origen de la teoría del caos |
| Rössler | a, b, c | Química, biología, osciladores |
| Halvorsen | a | Sistemas simétricos, física de partículas |
| Thomas | b | Dinámica cíclica disipativa |

## Features

- Integrador **RK4** real (no Euler)
- **Exponente de Lyapunov** estimado en tiempo real
- **Órbitas múltiples simultáneas** (1–20) con spread configurable
- **Diagrama de bifurcación** reactivo al parámetro ρ
- **Mapa logístico** con explicación inline
- **Web Audio** — tono modulado por velocidad orbital
- **Modo cinematográfico** fullscreen (tecla F)
- **Compartir URL** — encode completo de parámetros
- **Galería** de 6 configuraciones preseleccionadas
- **Exportación CSV** de datos de trayectoria
- Corre en cualquier navegador moderno, sin instalación
